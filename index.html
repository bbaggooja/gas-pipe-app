🔵 도시가스 — 배관 관경 산출
도시가스 배관의 적정 관경을 산출합니다.

• 계산 공식
  - Pole 식: 저압 배관 (일반적으로 가장 많이 사용)
  - Cox 식: 저압 배관 (Pole 식과 유사, 계수 차이)
  - Weymouth 식: 중·고압 배관

• 기본설정
  - 시설명, 공급압력(P0), 허용 압력강하(dH), 환산유량 설정
  - Weymouth 식 선택 시 종점압력(P2), 가스온도(T), 압축계수(Z), 효율계수(E) 추가 입력

• 사용량 합계 계산
  - 가스 기기별 용량(kcal/hr 또는 kW)과 수량 입력
  - 자동으로 m³/hr 환산유량 계산
  - [Q 적용] 버튼으로 구간입력의 유량(Q)에 자동 입력

• 구간 입력
  - 구간명, 용도/설명, 분기점 연결 설정
  - 유량(Q), 실제길이(L), 관종(배관 호칭경) 선택
  - 피팅류(엘보, 티, 밸브 등) 수량 입력 → 환산길이 자동계산
  - 유효길이 = 실제길이 + 피팅 환산길이

• 결과
  - 구간별 계산 관경, 선정 관경, 압력강하, 잔여압력 표시
  - 전체 판정(OK/NG) 및 간선 배관 흐름도
  - PDF/인쇄 및 CSV 내보내기 지원
🔥 LPG — 사용량 계산
LPG(프로판/부탄) 사용시설의 월 사용량, 저장능력, 압력조정기 용량 등을 계산합니다.

• 가스 종류 선택
  - 프로판(C₃H₈): 밀도 0.508 kg/L, 발열량 24,000 kcal/kg
  - 부탄(C₄H₁₀): 밀도 0.584 kg/L, 발열량 29,500 kcal/kg

• 기기 목록
  - 가스기기별 용량(kcal/hr)과 수량을 입력
  - 시간당 총 소비량(kg/hr, m³/hr) 자동 합산

• 탱크 용적 계산
  - 탱크 용적 V(L) 입력 → 환산용량(kg) 자동계산 (소수점 3자리)
  - 저장계수: 지상탱크 0.85 / 지하탱크 0.90
  - [적용] 버튼으로 아래 저장능력에 자동 반영

• 월 사용량 · 저장능력
  - 1일 사용시간 × 30일 = 월 사용량
  - 저장능력 = 계수 × 밀도 × V(L) × 탱크수량
  - 충전주기 = 월사용량 ÷ 가용저장량(잔액 제외)

• 최대 가스소비량 · 압력조정기 용량
  - 세대당 소비량 × 세대수 × 동시사용률
  - 압력조정기 용량 = 최대소비량 × 1.2 (안전율 20%)
🏭 고압 / 액화가스 — 저장능력 산정
고압가스안전관리법에 따른 저장능력을 산정합니다.

• 압축가스
  - W = (10P + 1) × V ÷ (10 + 1)
  - P: 최고충전압력(MPa), V: 내용적(L)
  - 결과: 저장능력(m³)

• 액화가스
  - W = V × C × d
  - V: 내용적(L), C: 정수(0.9), d: 비중(액비중/물비중)
  - 결과: 저장능력(kg)

• 적용 법규
  - 고압가스안전관리법 시행규칙
  - KGS FP111/FP112/FP211/FP212 등
⚖️ 단위환산
가스설비 관련 단위를 상호 변환합니다.

• 압력 단위
  - kPa ↔ mmH₂O ↔ atm ↔ bar ↔ psi ↔ MPa ↔ kgf/cm²

• 유량 단위
  - m³/hr ↔ L/min ↔ CFM ↔ GPM

• 열량 단위
  - kcal ↔ kJ ↔ BTU ↔ kW·h

  값을 입력하면 모든 단위로 실시간 변환됩니다.

  - 아음속 누출(P ≤ 임계압력비): 일반 유량 계산

• 결과
